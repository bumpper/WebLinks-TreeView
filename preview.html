Brave
  1. Open Brave browser.
  2. Click the three-line menu in the top-right corner.
  3. Select Bookmarks > Bookmark manager.
  4. Click the three-dot menu in the Bookmark manager.
  5. Select Export bookmarks and save the file.
Chrome
  1. Open Chrome browser.
  2. Click the three-dot menu in the top-right corner.
  3. Select Bookmarks > Bookmark manager.
  4. Click the three-dot menu in the Bookmark manager.
  5. Select Export bookmarks and save the file.
Edge
  1. Open Microsoft Edge browser.
  2. Click the three-dot menu in the top-right corner.
  3. Select Favorites > Manage favorites.
  4. Click the three-dot menu in the Favorites manager.
  5. Select Export favorites and save the file.
Firefox
  1. Open Firefox browser.
  2. Click the Library icon or open Bookmarks from the menu.
  3. Select Show All Bookmarks (or Manage Bookmarks).
  4. Click Import and Backup at the top.
  5. Select Export Bookmarks to HTML or Backup to JSON.
Opera
  1. Open Opera browser.
  2. Click the heart icon in the sidebar or go to Bookmarks.
  3. Click the Import/Export button.
  4. Select Export bookmarks and save the file.
Safari
  1. Open Safari browser.
  2. In the menu bar, click File > Export Bookmarks...
  3. Choose a location and save the bookmarks file.
