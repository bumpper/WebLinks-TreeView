Brave

  1. Open the Brave browser.
  2. Click the Hamburger menu (three-lines) in the top-right corner.
  3. Then select Bookmarks > Bookmark manager.

  4. Click the three-dot menu in the Bookmark manager.

  5. Select Export bookmarks and save the file.

Chrome

  1. Open the Chrome browser.
  2. Click the Breadcrumb menu (three-dots) in the top-right corner.
  3. Select Bookmarks > Bookmark manager.

  4. Click the Breadcrumb menu (three-dots) in the Bookmark manager.

  5. Select Export bookmarks and save the file.

Edge

  1. Open the Microsoft Edge browser.
  2. Click the Breadcrumb menu (three-dots) in the top-right corner.
  3. Select Favorites

  4. Click the three-dot menu in the Favorites menu > Export Favorites and Save as an HTML file.

Firefox

  1. Open the Firefox browser.
  2. Click the Hamburger menu (three-lines) > Bookmarks

  3. Select Manage Bookmarks

  4. Select Import & Backup > Export Bookmarks to HTML.

Opera

  1. Open the Opera browser.
  2. Click the Opera button in the top Left corner > Bookmarks > Manage Bookmarks...

  3. Click File > Export As HTML... button.

  4. Select Export bookmarks and Save as an HTML file.
Safari

  1. Open the Safari browser.
  2. In the menu bar, click File > Export Bookmarks...

  3. Choose a location and Save as an HTML file.
